# Pixel Tui Website

Static Pixel Tui website source. This repo is intended to be connected to the Pixel Tui Vercel project so the site is recoverable from GitHub.

## Vercel Settings

- Framework preset: Other
- Build command: leave blank or `npm run build`
- Output directory: `.`
- Install command: `npm install`

## Special Routes

`/example/anco` is rewritten to the ANCO Studio example app:

```text
https://salon-consultation-helper-v1.vercel.app/
```
